What this lookup tells you

  • Valid certifications show the holder's tier (Bronze, Silver, Gold, or Platinum), final score, number of questions correct, and the date of issue
  • Revoked certifications are clearly marked as no longer valid, with the reason if recorded
  • Unknown serials return a definitive "not found" result — no GSU certification carries that number
  • All certifications are anonymous — GSU records the credential, not the learner's identity. Anyone presenting the serial can prove the achievement; nobody can be tracked by it
  • Permanent and immutable — once issued, certification records cannot be altered. Only revocation status can change
